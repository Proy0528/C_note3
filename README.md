## 2024/04/14 : 조건문
#### 조건이란 : 맞거나 ```(true)``` 틀리거나 ```(false)``` 로 답변할 수 있는 질문
#### false : 0
#### true : 1
#### 0이 아닌 나머지 숫자들은 다 true로 인식
#### 그러나 컴퓨터에게 true를 표현하라 : 항상 1로 표현한다
----
## 비교
    <=, >=, ==, !=
    차례대로 -> 크거나 같다 / 작거나 같다 / 같다 / 같지 않다
